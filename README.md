> # AMA ChatBot

#### Web ChatBot using Flask and [ChatterBot](https://github.com/gunthercox/ChatterBot).

## Guide:
 1. Install Python, Flask, SQLAlchemy, and ChatterBot (either manually, or run `pip install -r requirements.txt`).
 2. Run `python app.py`.
